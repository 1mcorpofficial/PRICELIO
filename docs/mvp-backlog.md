# MVP backlog

## Must have
- PWA core screens: map, search, basket, receipt scan.
- Source badges with validity dates.
- Flyer ingest for 1-2 store chains.
- Receipt upload and extraction pipeline (basic).
- Basket optimizer for 1 store.
- Minimal admin tools: source health, extraction failures, low confidence list.

## Should have
- Receipt confirmation UI for low confidence lines.
- Basic product matching with alias table.
- Price history per product (median + last known).
- Map filters (category, verified, distance).

## Later (post-MVP)
- 2-store optimizer with travel cost.
- ShelfSnap and barcode mode.
- Personalized alerts.
- SEO city/category pages.
