# UI screens and flow

## Core screens
- Home / city feed: daily deals, new flyers, price shocks.
- Map: store pins, top deals, filters.
- Search: product detail with price history and sources.
- Basket: list to optimized plan (1/2 stores, travel tradeoff).
- Scan: receipt capture with quality checks.
- Report: overpaid items and savings summary.
- Profile: preferences, alerts, data controls.

## Three modes (single switch)
- Save today: map and top deals by city.
- Plan a purchase: basket optimizer with clear plan.
- Just bought: receipt scan and report.

## Price badges
- Each price shows source badge (FLYER/ONLINE/RECEIPT), validity, and verified status.
