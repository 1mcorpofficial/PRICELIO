# User profiles and scenarios

## Primary profiles
- Everyday saver: wants a simple "cheapest now" answer.
- Family: large baskets and project baskets.
- DIY: high ticket items, specs and unit price matter.
- Beauty/pharmacy: variants and mg/ml units matter.
- Data-driven: wants history and verified proof.

## Five main scenarios
- After purchase: scan receipt, see overpaid items.
- Before purchase: enter basket, get 1-2 store plan.
- Quick deal: open map and see top deals nearby.
- In-store: scan barcode or shelf label.
- Alerts: notify when a favorite item drops.

## Guest mode
- Allow short-term use with limits (map, few searches, 1-2 baskets).
- Prompt account creation only after limit reached.
