# KPIs

## User value
- Average savings per user per month (EUR).
- Receipts per active user.
- Basket optimization run -> accept rate.
- Retention D1/D7/D30.

## Data quality
- Offer coverage by city/category.
- Verified ratio.
- Dispute rate.
- Extraction success rate (flyers and receipts).

## Business
- Organic traffic to SEO pages.
- Premium conversion rate (later).
- B2B leads (later).
