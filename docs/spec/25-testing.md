# Testing and QA

## Test levels
- Unit tests for parsers, unit conversion, score logic.
- Integration tests for API flows (basket, receipt, map).
- Golden samples for receipts by store chain.
- Ingest regression samples for flyer templates.

## Data quality tests
- Price sanity and anomaly checks.
- Unit price correctness.
- Duplicate/merge detection for products.
- Offer validity and expiry checks.
