# Operations and monitoring

## Metrics
- Connector success rate and last successful fetch.
- Queue lag and backlog length.
- Receipt processing time (p50/p95).
- AI cost per receipt/page.
- API error rate (4xx/5xx).
- Slow DB queries.

## Alerts
- Connector down for > 2 hours.
- Expected count deviation.
- Queue lag threshold.
- AI provider timeouts or spikes.
- Security signals (failed logins, upload spikes).
