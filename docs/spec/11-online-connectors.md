# Online connectors

## When to use
- Public price is visible without login.
- Between flyer periods to keep data fresh.
- Long-tail coverage.

## Caching and limits
- Per URL/product TTL (6-24h).
- Rate limits per domain.
- Fallback to last successful cache with updated-at label.

## Offer vs real paid
- Online is always labeled ONLINE.
- Verified only when receipts confirm.
- Conflicts are shown clearly.

## Legal safety
- Prefer official feeds and partnerships.
- Scraping only public data with strict limits.
- Always store source_url.
