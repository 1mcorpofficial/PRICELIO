# Launch plan

## MVP scope
- Mobile-first PWA with map, search, basket, receipt scan.
- Flyer ingest for 1-2 store chains.
- Clear source badges and validity dates.
- Basket optimization for 1 store.
- Minimal admin panel (extraction failures, low confidence, source health).

## Beta
- Limited audience for fast feedback.
- Report a problem button.
- Receipt upload incentives (challenges, pro days).

## Post-MVP
- 2-store optimization with travel cost.
- ShelfSnap and barcode mode.
- More chains and cities.
- SEO city/category pages.
